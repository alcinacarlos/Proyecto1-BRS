1.1 Contraseña de usuario o Power-On

El objetivo es evitar el acceso no autorizado al equipo antes del arranque del sistema operativo.

Pasos

Reinicia el ordenador y accede al menú BIOS/UEFI (usualmente presionando Supr, F2, Esc o F10 según el fabricante).

Dirígete al apartado Security o Seguridad.

Busca la opción User Password, Power-On Password o similar.

Introduce una contraseña segura que combine letras, números y símbolos.

Guarda los cambios y reinicia el sistema.

El sistema pedirá una contraseña antes de iniciar el arranque.

1.2. Contraseña de administrador

Objetivo

Restringir el acceso y las modificaciones a la configuración de la BIOS/UEFI.

Pasos

Ingresa nuevamente al menú BIOS/UEFI.

En la sección Security, selecciona Administrator Password o Supervisor Password.

Crea una contraseña diferente de la de usuario, preferiblemente más compleja.

Guarda los cambios y reinicia el equipo.

Resultado esperado: Solo quien conozca la contraseña de administrador podrá modificar la configuración del BIOS/UEFI.

1.3. Arranques externos

Objetivo

Evitar que el sistema se inicie desde dispositivos externos (USB, DVD, etc.) que puedan contener software malicioso.

Pasos

Accede al BIOS/UEFI.

Busca la sección Boot, Boot Options o Advanced Boot Options.

Desactiva la opción Boot from external devices, USB Boot o Removable Media Boot.

Guarda los cambios.

✅ Resultado esperado: El equipo no podrá arrancar desde medios externos sin autorización previa.

1.4. Orden de arranque

Objetivo

Asegurar que el sistema operativo principal sea el primero en arrancar y evitar manipulaciones.

Pasos

Entra al BIOS/UEFI y localiza la pestaña Boot.

Establece como primera opción de arranque el disco donde está instalado el sistema operativo.

Mueve cualquier otro dispositivo (USB, CD/DVD, red, etc.) al final de la lista.

Guarda los cambios.

✅ Resultado esperado: El sistema solo iniciará desde el disco autorizado.

  1. Otras opciones de seguridad

Objetivo

Aplicar configuraciones adicionales que fortalezcan la seguridad general del BIOS/UEFI.

Recomendaciones

Activar Secure Boot: Garantiza que solo se carguen sistemas operativos y controladores firmados digitalmente.

Desactivar puertos innecesarios: Deshabilita puertos como FireWire, Thunderbolt o incluso USB si no se necesitan.

Habilitar TPM (Trusted Platform Module): Recomendado para cifrado de disco (BitLocker u otros).

Configurar el modo de arranque: Usar UEFI Mode en lugar de Legacy Mode para una mayor protección.

✅ Resultado esperado: El BIOS/UEFI cuenta con medidas adicionales que protegen la integridad y autenticidad del sistema.

Conclusión

Siguiendo estos pasos, la BIOS/UEFI quedará configurada de forma segura, cumpliendo con todos los criterios del nivel Bien (2 puntos) en cada indicador. Esto refuerza la protección del sistema frente a accesos no autorizados y manipulaciones en el arranque.
